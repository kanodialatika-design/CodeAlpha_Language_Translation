# 🌍 Language Translation Tool

A simple and user-friendly Language Translation Tool developed using **Python Flask**, **HTML**, **CSS**, and **JavaScript**.

This project was created as **Task 1** for the **CodeAlpha Artificial Intelligence Internship**.

---

## 📌 Features

- Translate text between multiple languages
- Auto detect source language
- User-friendly interface
- Responsive design
- Copy translated text with one click
- Fast translation using LibreTranslate API
- Error handling for invalid input

---

## 🛠️ Technologies Used

- Python
- Flask
- HTML5
- CSS3
- JavaScript
- LibreTranslate API

## 🚀 Installation

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/CodeAlpha_Language_Translation.git
```

### 2. Open the Project Folder

```bash
cd CodeAlpha_Language_Translation
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Run the Project

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000
```
